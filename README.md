# 個人財務管理系統 (Personal Finance System)

這是一套使用 **Python + Streamlit** 開發的個人財務管理系統。本系統旨在幫助使用者有效紀錄每日收支、視覺化消費結構，並透過數據分析達成理財目標。

---

## 核心功能說明

### 1. 使用者驗證系統

* **安全登入**：實作多使用者註冊與登入機制，確保財務資料的私密性。
* **密碼保護**：採用 **SHA-256** 加密演算技術，將密碼進行雜湊處理，不以明文儲存。

### 2. 收支管理與分類

* **紀錄功能**：允許使用者輸入日期、金額、收支類型（收入/支出）。
* **預設類別**：包含食、衣、住、行、育、樂及薪資獎金等。

### 3. 財務目標追蹤

* **目標設定**：使用者可設定每月的儲蓄目標金額。
* **進度可視化**：系統自動計算儲蓄缺口，並以**動態進度條**呈現達成率。

### 4. 視覺化分析報表

* **圖表分析**：透過 **Plotly** 呈現支出比例圓餅圖與收支趨勢長條圖。

---

## 技術架構與套件

* **程式語言**: `Python 3.9+`
* **網頁框架**: `Streamlit`
* **資料處理**: `Pandas`
* **圖表繪製**: `Plotly Express`
* **資料庫**: `SQLite` (實現資料持久化儲存)

---

## 安裝與執行方式

### 1. 建立並啟動虛擬環境

請在 PowerShell 中依序輸入以下指令：

python -m venv venv
.\venv\Scripts\Activate.ps1

### 2. 安裝必要套件

pip install streamlit pandas plotly

### 3. 啟動系統

streamlit run main.py

---

## 資料庫設計說明

本專案使用 SQLite 資料庫進行持久化儲存，表格結構如下：

| 資料表名稱 | 主要功能 | 備註說明 |
| :--- | :--- | :--- |
| **users** | 儲存使用者帳號 | 密碼以 SHA-256 雜湊值儲存 |
| **transactions** | 儲存收支明細 | 紀錄日期、類型、類別、金額 |
| **goals** | 儲存儲蓄目標 | 紀錄每個使用者設定的儲蓄金錢目標 |
